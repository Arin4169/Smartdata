# 네이버 스마트 스토어 데이터 분석 웹앱

네이버 스마트 스토어에서 제공하는 판매 데이터와 리뷰 데이터를 분석하여 시각화하는 완성된 웹 애플리케이션입니다.

## ✨ 주요 기능

### 🏠 홈페이지
- 4개의 분석 카드로 구성된 직관적인 인터페이스
- 각 분석 유형별 설명과 기대 효과 제공
- 반응형 카드 디자인으로 사용자 경험 최적화
- 브랜드 메시지와 사용 안내 섹션 포함

### 📖 데이터 분석 사용안내
- **Smart Data 확장 프로그램 설치 가이드**: Chrome 웹스토어 링크 및 설치 방법
- **데이터 다운로드 방법**: 스마트 스토어에서 각 데이터 타입별 다운로드 안내
- **파일 업로드 및 분석 가이드**: 단계별 사용 방법 상세 설명
- **자주 묻는 질문 및 문제 해결**: 사용자 지원 정보

### 📊 리뷰 분석 - 워드클라우드
- 리뷰 텍스트를 분석하여 워드클라우드 형태로 시각화
- 언급량이 많은 상위 20개 단어 추출 및 시각화 (높은 빈도순 정렬)
- **불용어 관리 시스템**: 
  - 실시간 불용어 추가/삭제 기능
  - 텍스트 입력 방식으로 불용어 관리
  - 엔터키 입력으로 즉시 적용
  - 기본 불용어 포함한 맞춤형 필터링

### 😊 리뷰 분석 - 감정분석
- 리뷰 텍스트의 감정 분석 (긍정/중립/부정)
- 감정별 분포 막대 그래프 및 파이 차트 시각화
- **감정별 카테고리 분석**:
  - 긍정/중립/부정 리뷰 각각의 카테고리별 분석
  - 카테고리별 언급 빈도 시각화
  - 적응형 막대 폭으로 데이터 양에 따른 최적화
- **수치 해석 안내**: 차트 해석 방법 및 카테고리별 집계 설명
- 대형 탭 폰트로 가독성 향상

### 🎯 옵션 분석
- 판매량이 가장 많은 상위 10개 옵션 추출 및 시각화
- 데이터 테이블과 막대 그래프로 이중 시각화
- 옵션별 판매량 순위 및 비교 분석

### 📈 스토어 전체 판매현황
- **4가지 종합 분석 제공**:
  - 🏆 매출 랭킹: 기간별 매출 상위 10개 상품
  - ⚡ 매출 효율성: 가격 대비 매출 성과 분석
  - 💰 가격대별 분석: 동적 가격대 설정으로 4구간 분석
  - 💡 리뷰-매출 인사이트: 숨겨진 보석, 잠재력 미달 상품 발굴
- 기간별 분석 지원 (1개월, 3개월, 6개월, 1년)
- 매출 요약 통계 제공

## 🚀 사용 방법

### 1. 환경 설정
```bash
# 필요한 라이브러리 설치
pip install -r requirements.txt

# 또는 배치 파일 실행 (Windows)
run.bat
```

### 2. 애플리케이션 실행
```bash
streamlit run app.py
```

### 3. 데이터 사용 방법
- **샘플 데이터**: 파일 업로드 없이 바로 분석 체험 가능
- **파일 업로드**: 좌측 사이드바에서 데이터 파일 업로드 (최대 3개 파일)
- **자동 감지**: 파일명과 컬럼을 분석하여 파일 유형을 자동으로 감지

### 4. 분석 수행
- 좌측 사이드바에서 원하는 분석 유형 선택
- 각 분석 결과는 실시간으로 업데이트
- 분석 가이드를 참고하여 결과 해석

## 📁 필요한 데이터 파일 형식

### 1. 리뷰 분석 파일 (reviewcontents)
- **형식**: Excel (.xlsx) 또는 CSV (.csv)
- **필수 컬럼**: `REVIEW_CONTENT`, `review_content`, `리뷰내용`, `내용`, `CONTENT` 중 하나
- **자동 감지**: 파일명에 'review' 포함 시 자동 인식

### 2. 옵션 비율 파일 
- **형식**: Excel (.xlsx) 또는 CSV (.csv)
- **필수 컬럼**: 
  - 옵션: `OPTION_INFO`, `option_info`, `옵션정보`, `옵션명`, `상품옵션` 중 하나
  - 수량: `COUNT`, `count`, `수량`, `판매량`, `판매수량` 중 하나
- **자동 감지**: 파일명에 '옵션' 포함 시 자동 인식

### 3. 판매 현황 파일 (스토어전체판매현황)
- **형식**: Excel (.xlsx) 또는 CSV (.csv)
- **필수 컬럼**: `상품명`, 매출 관련 컬럼들
- **자동 감지**: 파일명에 '판매현황' 또는 '스토어' 포함 시 자동 인식
